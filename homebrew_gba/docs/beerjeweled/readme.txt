                         - BEERJEWELED -

        An entry for the Interactive Compo @ Evoke 2013
________________________________________________________________

Remember: Never drink mixed, but stay with one drink. Otherwise
you'll have a terrible hangover the next morning!
Swap two drinks on the playfield in order to make a set of three
or more identical beverages.

________________________________________________________________

System Requirements:
________________________________________________________________

Nintendo Gameboy Advance (SP), Gameboy Micro or DS (Lite) or
Nintendo Gamecube with Gameboy Player or
PC/Mac/Tablet/Phone/Thing running a GBA emulator

We recommend real hardware.

Controls:
________________________________________________________________

Joypad ............................................. Move cursor
A + Joypad ......................................... Swap drinks
START ...................................... Pause / resume game

________________________________________________________________

Gameplay:
________________________________________________________________

Move your cursor to one of the drinks, then press Button A and
a direction on the joypad to swap it with the adjacent beverage.
This will only work if you create a set of 3 or more identical
drinks, which will disappear, and the tiles beneath will turn 
from grey to golden. New beverages will fall on the playfield.
The game continues to the next level when all background tiles
are golden, and end if you run out of time.

________________________________________________________________

Scoring:
________________________________________________________________

Removing 3 items with a single move ................... 100 Pts.
Removing 4 items with a single move ................... 250 Pts.
Removing 5 or more items with a single move ........... 500 Pts.

A time bonus is awarded after a level is finished.
